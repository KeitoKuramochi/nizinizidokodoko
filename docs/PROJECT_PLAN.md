## アプリの目的

「虹が出てるよ」という情報を受け取ったとき、自分のいる場所から虹がどの方角にあるかを即座に確認できるようにする。
太陽位置の計算と気象データを組み合わせて、「今まさに虹が見える方角」をコンパスで示す。

## 対象ユーザー

- 屋外にいるスマートフォンユーザー
- 虹が出ているという情報を得て、どちらを向けば見えるか知りたい人
- ログインや事前設定なしにすぐ使いたい人

## ユーザーが達成できること

- アプリを開いて位置情報を許可するだけで、虹が見える方角をコンパス（矢印）で確認できる
- 「今日は虹が出やすい気象条件か」を一目で把握できる
- 条件が揃っていない場合はその理由を確認できる

## MVP の範囲

- 現在地GPS取得（ブラウザ Geolocation API）
- SunCalc.js による太陽方位角・仰角のリアルタイム計算
- 反太陽点（太陽の真反対方向）から虹の方角を算出して表示
- Open-Meteo API で現在地の降水量・雲量を取得
- 気象条件の判定（「虹が出やすい」「出にくい」）とその理由の表示
- コンパス形式または方位矢印での虹方角の表示
- スマートフォン幅（375px〜）での動作
- 単一画面アプリ（画面遷移なし）

## 将来機能（MVP 対象外）

- ユーザー投稿・虹目撃情報の共有
- SNS（X/Twitter等）連携
- 過去の虹情報の履歴
- プッシュ通知（「今虹が出やすい」アラート）
- デバイスコンパスセンサーとの連動（物理コンパス）

## 技術スタック（決定済みのみ）

- Next.js + TypeScript + Tailwind CSS（既存プロジェクト構成に合わせる）
- SunCalc.js（太陽位置計算ライブラリ、npm パッケージ）
- Open-Meteo API（無料・APIキー不要の気象API）
- ブラウザ Geolocation API（位置情報取得）
- バックエンドなし（フロントエンドのみ）
