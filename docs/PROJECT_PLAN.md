# PROJECT_PLAN.md

## アプリ名
**Nizi Pro（ニジプロ）**

## 目的
「虹が出てるよ！」という情報を受け取っても、自分の位置からどの方向に虹があるのか・建物や雲で隠れているのかがわからない問題を解決する。
現在地から虹の方角をリアルタイムで提示し、虹を見つける体験を楽しくする。

## 対象ユーザー
- スマートフォンを持ち歩く一般ユーザー（10〜40 代）
- SNS で「虹出てる！」という投稿を見て外に飛び出したくなる人
- 天気・自然現象に興味がある人

## コアバリュー
「今すぐ虹を見られるか」が 1 タップでわかる

## MVP（最初に作るもの）
1. ブラウザの Geolocation API で現在地を取得
2. モック／ハードコードした虹の発生位置（緯度経度）を用意
3. 現在地 → 虹の位置の方位角（bearing）を計算
4. コンパス風 UI で「北から何度」または「東・南東・北北西」などの方角を表示
5. 「虹の目撃情報なし」状態の空ステートも表示

## 将来機能（MVP 後）
- X (Twitter) API で「虹」ツイートを収集し、投稿者の位置情報から虹の位置を推定
- 複数の目撃情報をクラスタリングして精度向上
- AR カメラ表示（スマホのカメラを向けると虹の方向にインジケーター）
- プッシュ通知「今近くで虹が出ています！」
- 虹の持続時間の推定
- 過去の虹出現マップ

## 技術的方針
- 外部 API キーが不要な機能から始める（Geolocation のみ）
- 位置情報はブラウザ側で完結させ、サーバーに送信しない（プライバシー配慮）
- PWA 対応（将来）でホーム画面追加を可能にする
